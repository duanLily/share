# H5中websocket的应用
  ## HTTP
    |-超文本传输协议（HTTP,HyperText Transfer Protocol）是互联网上应用最为广泛的一种网络协议。
      用于从WWW服务器传输超文本到本地浏览器的传送协议。
      1、建立连接（三次握手）
      2、发送请求（request headers）
      3、响应请求（response headers）
      4、断开连接（四次握手）
  ## Websocket
    Websocket protocol是HTML一种新的协议。它实现了浏览器与服务器全双工通信（full-duple）。
    一开始的握手需要借助HTTP请求完成。
